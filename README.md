# HotStone Center

Dashboard for HotStone-SEO

## Configuration

Update configuration at `.env` file. 
```
PORT=3000
CHOKIDAR_USEPOLLING=true
```

Please use `.env.template` if no `.env` available

---

## Development Guide

### Prerequisite

Install [NodeJS](https://nodejs.org/en/)
```bash
brew install node
```

### Install

```bash
npm install
```

### Run 

```bash
npm start
```
